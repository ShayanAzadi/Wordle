**Description:**  
This is a simple and fun implementation of the popular Wordle game using C# and Windows Forms. Players try to guess the hidden 5-letter word within 6 attempts. The game provides color-coded feedback for letters: green for correct letters in the correct position, yellow for correct letters in the wrong position, and gray for incorrect letters.

## Features
- Guess the hidden 5-letter word in 6 attempts.  
- Color-coded feedback for each letter.  
- Simple graphical interface using Windows Forms.  
- Uses a text file (`DATA.txt`) for storing words.
